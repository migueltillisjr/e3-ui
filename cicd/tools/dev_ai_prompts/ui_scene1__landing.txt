**Objective**
Generate a single, production-ready **HTML file (one file, no external assets)** that mimics the **look & feel** of an existing website while keeping content original (paraphrased). The page must feature two primary buttons and remember the user’s preference to launch the AI agent next time.

**Inputs (variables):**

* `{{SOURCE_URL}}` – the public website to **learn** visual style + voice from.
* `{{AI_AGENT_URL}}` – destination for “Launch AI Agent”.
* `{{SITE_URL}}` – destination for “Visit Site” (usually the same as `{{SOURCE_URL}}`).
* `{{BRAND_NAME}}` – text label used in the eyebrow and footer (e.g., “Info Pioneer”).

**Style & Branding (learned from {{SOURCE_URL}}):**

1. **Extract & approximate brand tokens** from `{{SOURCE_URL}}`:

   * Primary/secondary colors, background style (e.g., gradient, solid, subtle texture).
   * Heading & body font families (or closest web-safe equivalents).
   * Button corner radius, shadow depth, spacing rhythm, and general vibe (minimal, bold, glossy, etc.).
2. **Do not copy proprietary images, logos, or trademarks.** Mimic the **style**, not the exact assets.
3. Ensure **WCAG-friendly contrast**, visible focus states, and reduced-motion support.

**Layout (required):**

* Centered card/hero with:

  * Eyebrow label: `{{BRAND_NAME}}`
  * H1 headline (you may synthesize a concise, punchy line that matches the brand voice).
  * Subheadline (1 short sentence, paraphrased from {{SOURCE_URL}} tone).
* **Two large buttons side-by-side** (stack on mobile ≤640px):

  * Left (accent): **Launch AI Agent** → `{{AI_AGENT_URL}}`
  * Right (neutral): **Visit {{BRAND_NAME}}** → `{{SITE_URL}}`
* **Brief summary** (2–4 sentences) **paraphrasing** what the source site offers (language + branding), under the buttons. No copy/paste; keep it original and concise.
* Optional KPI chips: if the source communicates clear outcome metrics, include up to 3 compact chips; otherwise omit.
* Footer with © year and a link to `{{SITE_URL}}`.

**Behavior (must implement):**

* When the user clicks **Launch AI Agent**, **store preference** in `localStorage` (key: `ip_launch_pref = "agent"`).
* On return visits, if preference exists **and** no `?nopref=1` query param:

  * Show a brief notice (polite ARIA status), then **auto-redirect** to `{{AI_AGENT_URL}}` (use `location.replace`).
* Provide a **“Change preference”** link that clears the saved choice.
* Keyboard and screen-reader friendly; visible focus rings.

**Technical constraints:**

* **One HTML file** only; embed `<style>` and `<script>` in the document.
* No external fonts, CSS, or JS. Use system fonts or safe fallbacks if needed.
* Use CSS Grid/Flex for layout; use `clamp()` for responsive type/spacing.
* Respect `prefers-reduced-motion`.
* Add meta tags: `theme-color`, `color-scheme`, and a short `description`.

**Deliverable**
Return **exactly one** HTML document. Use this structure and **inject the learned tokens** (colors, radii, shadows, typography) to reflect `{{SOURCE_URL}}`:
