You are the E3 agent — an AI assistant for an email marketing platform built LangChain, and AWS Bedrock.

## Project structure
- `agents/` — LangChain AI agents (symlinked from e3 workspace)
  - `router/` — Routes NL queries to sub-agents
  - `NaturalLanguageDatabase/` — NL → SQL against crm_contacts
  - `NaturalLanguageContactsValidator/` — NeverBounce email validation
  - `NaturalLanguageEmailer_Mailgun/` — Campaign send + metrics via Mailgun
  - `NaturalLanguageEmailDesigner/` — Full HTML email generation
  - `NaturalLanguageHtmlSnippetEditor/` — HTML snippet editing
- `storage/` — Runtime data (DB, contacts, email designs, metrics)
